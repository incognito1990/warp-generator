config generator

WARP · AmneziaWG

Generate a ready-to-use AmneziaWG config with optional obfuscation. Keys are created on the fly for each request — nothing is stored on the server.

AmneziaWG clients: Windows macOS Linux Android iOS
step 1 — mode
ℹ Zero Trust requires a JWT from your organization's Cloudflare Access portal. Use email OTP or paste a token manually.
⚠ Tokens expire in about one minute. On the portal page open DevTools → Console and run: console.log(document.querySelector("meta[http-equiv='refresh']").content.split("=")[2])
✓ Consumer mode: no login required. The server registers a free WARP device with Cloudflare automatically.
step 2 — DNS
ℹ Cloudflare DNS is used by default. Change only if you need custom resolvers.
step 3 — routing (AllowedIPs)

All traffic goes through the VPN: 0.0.0.0/0 and ::/0 (when IPv6 is enabled).

Enter subnets to preview AllowedIPs…
step 4 — advanced (optional)
Device will be named: warp-‹xxxxx›

Uses the IPv6 endpoint Cloudflare assigns to this peer.

Endpoint will be: [2a14:7583:f764:64::162.159.193.1]:2408
ℹ Change this only if you know why. A custom domain generates a fresh QUIC Initial packet with that SNI.
